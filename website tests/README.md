Website test (Nova Drive)

Quick start (requires Bun or Node):

1. Install deps
   - bun: `bun install`
   - npm: `npm install`

2. Run dev server
   - bun: `bun run dev`
   - npm: `npm run dev`

The page calls the backend pricing API at `http://localhost:5000/api/pricing/estimate`. Ensure the API is running (docker-compose or `dotnet run`).
